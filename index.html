Prática com HTML

A aula de hoje é para podermos continuar com a crição do nosso site através do VSCODE dentro do GitHub

O objetivo dessa aula de hoje é: que vocês entendam a diferença do uso do HTML para o uso do CSS. Também temos como objetivo, mudar a apresentação visual dos textos que temos.

A intenção em primeiro lugar é que aprendam o conteúdo e na prática, formalize mais ainda.
